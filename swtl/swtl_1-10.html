Chapter 1: Alerts

Alerts: A box that pops with a message.
alert("Hi Julie!!!");
Displays: Hi Julie!!!

Chapter 2: Variables for Strings

Variables: A word or number that holds a value
var name = "Julie";
name = "Julie";

Chapter 3: Variables for numbers

Variables: A word or number that holds a value
var weight = 150;
weight = 150;

Chapter 4: VariableName Legal and Illegal

Variables names can not be a keyword.
Variable names can't have spaces
Variable names can have letters, numbers, $, and underscores.
Legal: var myName;
Illegal: var class;

Chapter 5: Math expressions:
Familiar operators

Arithmetic operators: +, -, *, /, %
var num = 1 + 5
or
alert(1 + 5)
both display 6

Chapter 6: Math expressions:
Unfamiliar operators

num++, num--, ++num, --num
var num = 5;
var newNum = num++;
Displays: 6

Chapter 7: Math expressions:
Eliminating ambiguity

The order of operations
var newNum = 1 + 5 * 6;
5 * 6 then add 1
newNum = 31;
