Chapter 25:
Strings:
Replacing characters

Use replace to replace the first instinct a word or number with another.
var x = "The rain in Spain";
x = x.replace("Spain", "France");
output: x = "The rain in France"

Use replace and / / g to global replace a word. var x = "The rain in Spain";
x = x.replace(/Spain/g, "France");
output: x = "The rain in France"

Chapter 26:
Rounding numbers

Use Math.round() to nearest integer.
var x = Math.round(1.5); // rounded up since it is .5 and a positive number. output: x = 2

Use Math.ceil() to round up.
var.x = Math.ceil(1.1);
output: x = 2

Use Math.floor() to round down.
var x = Math.floor(2.5);
output: x = 2

Chapter 28:
Converting strings to integers and decimals

Use parseInt to convert a number to an integer.
var x = parseInt("1.99");
output: x = "1"

Use parseFloat to preserve the decimal value of a decimal.
var x = parseFloat("1.99");
output: x = "1.99"

Chapter 29:
Converting strings to numbers, numbers to strings

Use Number to covert a string to a number.
var x = "24"
var y = Number(x);
output: y = 24

Use toString() to covert a number to a string
var x = 24;
var y = x.toString();
output: y = "24"

Chapter 30:
Controlling the length of decimals

Use toFixed() to help control the number of decimals.
var x = 1.234567;
x = x.toFixed(2);
output: x = "1.23"
